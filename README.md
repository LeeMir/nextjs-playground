# Gatsby Playground

NextJS 공부용 Repo.

## Install

### Environment

> OS : Windows10 x64, macOS 12.3.1
> Node : v16.14.2(LTS)
> Yarn : v1.22.18

### Dependencies

```bash
yarn install # 또는 그냥 yarn
```

## Run

```bash
yarn start
```
